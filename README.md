# Projeto Sustentabilidade e Reciclagem

## Sobre o Projeto

Este projeto é um site educativo sobre sustentabilidade, reciclagem e
preservação do meio ambiente.

O objetivo é conscientizar os usuários sobre a importância de práticas
sustentáveis no dia a dia, oferecendo informações, dicas e recursos
interativos como quiz, busca de pontos de coleta e conteúdos educativos.

O site possui sistema de login e cadastro (simulado), formulário de
newsletter, páginas informativas e funcionalidades interativas
desenvolvidas em HTML, CSS e JavaScript.



## Como Usar

1.  Baixe ou clone o projeto.
2.  Abra o arquivo `index.html` em um navegador.
3.  Utilize o menu de navegação para acessar as páginas:
    -   Página Inicial
    -   Reciclagem
    -   Práticas Sustentáveis
    -   Mudanças Climáticas
    -   Login/Cadastro

### Funcionalidades disponíveis:

-    Cadastro na newsletter com validação de nome e e-mail.
-    Sistema de login e cadastro com validação de dados.
-    Sistema de abas para visualizar diferentes tipos de materiais
    recicláveis.
-    Quiz interativo sobre reciclagem.
-    Busca simulada de pontos de coleta através do CEP.
-    Informações educativas sobre mudanças climáticas e boas práticas
    ambientais.


##  Como Funciona

O projeto foi desenvolvido utilizando:

-   HTML → Estrutura das páginas.
-   CSS → Estilização e layout responsivo.
-   JavaScript → Funcionalidades interativas e validações.

### Principais funcionalidades técnicas:

-   Menu responsivo com botão hamburger.
-   Alternância entre formulário de login e cadastro.
-   Validação de formulários utilizando JavaScript e expressões
    regulares (Regex).
-   Quiz com sistema de pontuação e feedback automático.
-   Sistema de abas dinâmicas para exibir conteúdos diferentes.
-   Simulação de busca de pontos de coleta com validação de CEP.
-   Validação de senha com requisitos de segurança (letras maiúsculas,
    minúsculas e números).
